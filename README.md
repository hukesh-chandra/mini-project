# Mini Project

This repo is a collection of small projects, UI practice, and experiments.

It is not one single app. It is more like a workspace where I keep different mini builds in one place.


## How To Use This Repo

Each folder is different.

- for plain HTML, CSS, and JS folders, you can open the `index.html` file directly
- for folders that have their own setup, check that folder first and run it from there
- some subfolders already have their own `README.md`

## Why This Repo Exists

This repo helps me keep small ideas, practice work, and prototypes in one place.
Some folders are very simple, and some are more complete than others.

## Notes

- not every folder is a finished project
- some folders are quick experiments or design practice
- setup is not the same across all subprojects
